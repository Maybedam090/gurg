# 🚀 Quick Start Guide

## ⚡ Play Now (Locally)

1. **No installation needed!**
2. Open `stocks.html` in your web browser
3. Start trading immediately!

That's it! No server, no download, no setup.

---

## 📂 File Structure

```
your-folder/
├── stocks.html       (Main game file)
├── stocks.css        (Styling)
├── stocks.js         (Game logic)
├── README.md         (This file)
└── PUBLISHING_GUIDE.md (How to deploy)
```

**Make sure all 3 main files are in the SAME folder!**

## ✅ System Requirements

- **Any modern web browser** (Chrome, Firefox, Safari, Edge)
- **Internet connection** - Not required! Game runs 100% offline

   - Click any stock card to see analysis
   - Look for "📈 UPTREND" + "BULLISH" momentum

- Consider selling for profit

- Click another stock to see charts

- Make your second trade

## 🎮 Basic Commands

| Action | How | Result |
|--------|-----|--------|
| **View Stock Details** | Click stock card | Opens analysis modal |
| **Quick Buy** | In modal, enter quantity, click BUY | Purchase stock |
| **Buy from Menu** | Select + quantity → BUY button | Purchase stock |
| **Sell Stock** | Select owned stock → SELL button | Liquidate position |
| **Next Day** | Click "NEXT DAY" button | Time passes, prices change |
| **Get Loan** | Click "GET LOAN" in sidebar | Borrow money (15% interest/day) |
| **Repay Loan** | Click "REPAY LOAN" sidebar | Pay back borrowed money |
| **Sell All** | Click "SELL ALL" in sidebar | Liquidate entire portfolio |
| **Search Stocks** | Type in search box | Filter by name/ticker |
| **Filter Sector** | Select from dropdown | Show only one industry |
| **Toggle Menu** | Click ☰ on mobile | Show/hide sidebar |

## 💡 Beginner Tips

1. **Start Small** - Trade just 5-10 shares initially
2. **Watch the Trends** - Green = up, Red = down
3. **Use Recommendations** - Green "BUY" badges are good signals
4. **Don't Over-Leverage** - Don't borrow if unsure
5. **Diversify Early** - Buy stocks from different sectors
6. **Let Dividends Grow** - Hold good stocks for passive income
7. **Take Profits** - Sell winners after decent gains
8. **Cut Losses** - Don't hold losing positions too long

## 📊 Understanding the Numbers

### Cash vs Assets
- **Cash:** Money in hand (ready to spend)
- **Assets:** Total portfolio value (holdings + cash)

### Profit/Loss
- **Green number:** Making money! 📈
- **Red number:** Losing money! 📉

### Volatility
- **5%:** Stable (less risky)
- **15%:** Active (more movement)

### Dividend
- **0.01%:** Minimal income
- **0.07%:** Great income stream

## 🔧 Troubleshooting

**Game won't load?**
- Check all 3 files (html, css, js) are in same folder
- Try different browser
- Clear browser cache (Ctrl+Shift+Delete)

**Numbers look wrong?**
- Refresh page (Ctrl+R)
- Check you didn't accidentally sell all

**Can't see sidebar on mobile?**
- Click ☰ menu button
- Or rotate to landscape

**Stock prices frozen?**
- Click "NEXT DAY" button
- Prices only update daily

**Lost my data?**
- Refresh page = new game
- (Data saves only during session)
